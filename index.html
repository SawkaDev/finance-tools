Notes

Data & method

Returns: Ibbotson/Morningstar SBBI U.S. Large-Cap total return Inflation: BLS CPI-U (Dec-to-Dec)

Total return = dividends reinvested each year (the return an investor actually earned). Each "history" invests your plan at the start of a year and holds for the chosen number of years, using that era's real annual returns. Monthly contributions are added at the start of each month; because the source data is annual, each year's return & inflation are split geometrically across its 12 months. With “grows with inflation” selected, each deposit is the entered amount in start-year purchasing power — the nominal deposit rises along that history's own CPI path (like savings tracking a salary); with “Fixed $”, the nominal deposit stays the same forever.

Real strips out inflation and holds your starting stake at constant purchasing power, so every result is in today's dollars — what the money could actually buy now. Because the starting amount is the same for every history, the real results are directly comparable across start years (raw nominal dollars are not — each is inflated by a different amount over a different era). Nominal is the literal future statement number, before adjusting for inflation. Return/yr is the money-weighted return (IRR) — the single annual rate that grows every dollar, from the day it went in, into the final balance; with a pure lump sum it equals plain CAGR (end/start)^(1/years)−1. Growth × = ending value ÷ money put in — in Real mode both sides are inflation-adjusted (each contribution valued at the purchasing power it had when contributed), so it reads as a true purchasing-power multiple; "lost purchasing power" and red bars use the same inflation-adjusted baseline.

One statistical caveat: neighboring histories overlap (the 1950 and 1951 windows share 39 of 40 years), so the histories are not independent samples — counts like "3 of 61" describe how often something happened across start years, not 61 separate experiments.

Returns cross-verified against the independent Damodaran (NYU Stern) S&P 500 series (mean difference 0.38pp across 98 years). Inflation for 1926–1928 (−1.1%, −2.3%, −1.2%) and 2025 (+2.7%) from BLS.

Not investment advice. Past performance does not predict future results — showing the full range is the whole point.
